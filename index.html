Hospedagem grátis no GitHub Pages

Vantagens:

O GitHub Pages permite hospedar sites estáticos de forma gratuita, usando arquivos HTML,
CSS e JavaScript armazenados em um repositório do GitHub. É uma opção prática para portfólios, páginas de projetos,
documentação e pequenos sites pessoais, sem precisar pagar por um servidor tradicional.

Desvantagens:

Não é adequado para sites dinâmicos que requerem back-end ou interatividade complexa.
